ПМУ 9

ПМУ-9 Шатун это мобильная мишенная установка, предназначенная для стрельбы из всех носимых видов оружия по качающейся цели. В отличие от аналогов - имеет плавную регулировку скорости и амплитуды качаний - крепление мишени позволяет использовать ПМУ-9 для показа мишеней различных габаритов "из-за угла", в проемах и т.п. - система радиоуправления позволяет интегрировать установку в мишенную обстановку для совместной работы по единому сценарию. Опционально, возможна установка электронной мишени для пневматики №4 в виде металлического или пластикового щита 500х500мм, с передачей результата по сети WiFi на планшетный компьютер стрелка.
